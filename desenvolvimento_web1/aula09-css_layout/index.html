Criação de Layout

Conteúdo

O elemento main define o conteúdo principal em sua página. Entende-se como conteúdo principal aquele relacionado diretamente com o tópico central o qual deverá ser único na página, ou seja, dentro do elemento main não deverão ser incluidas seções da página que sejam comuns a todo o site, tais como mecanismo de navegação, informações de copyright e campos de busca.

Coluna

O elemento aside representa uma seção de uma página que consiste em conteúdo relacionado ao conteúdo do seu entorno. Essas seções são muitas vezes representadas como blocos laterais
